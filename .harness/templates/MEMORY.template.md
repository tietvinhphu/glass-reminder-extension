# Memory Index

> File này là **index** trỏ tới các memory files. Mỗi file memory = 1 fact/context quan trọng.
> Đọc file này đầu tiên mỗi session, sau đó load các files cần thiết.

## Cấu trúc

Mỗi file memory có:
- **YAML frontmatter** (name, description, type)
- **Body** markdown với nội dung cụ thể
- **Why + How to apply** lines cho feedback/project memories
- **Wiki-links** `[[other-memory-name]]` để liên kết

## Types

- `user` — ai là owner, cách học, preferences
- `project` — trạng thái dự án, checkpoints, deadlines
- `feedback` — quy tắc đã học được qua lỗi
- `reference` — pointers tới external resources
